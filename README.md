# Pyqt QR Code Cgenerator
A simple QR Code generator using a GUI build in PyQT5.

# Features 
- Save the generated QR Code by clicking on the image.
- QR Code image auto-update while typing.

# Installation 
> `pip install qrcode`
> `pip install pyautogui`
> `pip install PyQt5`
